Library Management Data Analysis (SQL)

Tech Stack: SQL

Web Scraping on Cars with Python

Employed Regular Expressions for identifying and extracting relevant classes, tags, features, and targets. Cleaned, preprocessed, and analyzed data using EDA techniques.

Tech Stack: Jupyter Notebook, Libraries: Requests, BeautifulSoup, NumPy, Pandas, Matplotlib, Regex, Seaborn
